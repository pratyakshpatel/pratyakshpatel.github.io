Isolation Forest

July 2025 | Project Link

We've all learnt and implemented random forest, its one of the first algorithms we learn while learning ML. Most of the times, we strive to find patterns using these algorithms, another paradigm that exists is that of anomaly detection; learning about it is how I came across Isolation Forests. In this project, I wrote a full isolation forest algorithm from first principles using numpy. The model constructs multiple binary trees by recursively partitioning randomly subsampled data, isolating anomalies with shorter path lengths. I applied this on synthetically generated 2d data (normal distribution + uniform outliers), computed anomaly scores based on average path length, and visualized the detection results with a threshold-based classification.
